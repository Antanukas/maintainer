# maintainer

Maintainer is a role which can be chose for some period. E.g for single 2 week sprint.
Maintainer is reposnsible for:
  1. making sure that no merge breaks build and fix it if it does.
  2. deploy to development environments
  3. investigate issues which raises in development or production
  4. TODO add more

## Prerequisites

You will need [Leiningen][] 2.0.0 or above installed.

[leiningen]: https://github.com/technomancy/leiningen

## Running

To start a web server for the application, run:

    lein ring server
